Список команд:

  • Друк (print) — друкування.
  • Якщо (if) — умовне виконання коду.
  • Функція (function) — блок коду для повторного використання.
  • Поки (while) — повторення, доки умова істинна.
  • Змінна (var) — змінна, значення якої можна змінити.
  • Константа (const) — незмінне значення.
  • Для (for) — цикл з лічильником.
  • Логічні оператори (та, або) — перевірка кількох умов.
  • Зупинити (break) — вихід із циклу.
  • Пропустити (continue) — пропускає одну ітерацію циклу.
  • Попередження (alert) — показує повідомлення.
  • ВипадковеЧисло (Math.random) — випадкове число від 0 до 1.
  • Дата (Date) — створення об’єкта дати.
  • Зараз (Date.now) — поточний час в мілісекундах.
  • Округлення — округлення чисел: round, floor, ceil.
  • Мінімум / Максимум — пошук найменшого або найбільшого значення.
  • Корінь — обчислення квадратного кореня з числа.
  • Степінь — піднесення числа до степеня.
  • Довжина — визначення довжини рядка або масиву.
  • Великими / Малими — зміна регістру символів у рядку.
  • ЦілеЧисло / Число — перетворення тексту в число.
  • Масив — створення та зміна списків.
  • Обєкт — робота з парою ключ-значення.
  • ЄКлюч / Видалити — перевірка та видалення ключів обєкта.
